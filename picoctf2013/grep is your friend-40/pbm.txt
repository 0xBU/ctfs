After plugging the robot into the computer, the robot asks for the name of a file containing the string SECRET AUTH CODE. You can find it using the command-line[1] interface in /problems/grep.tar or by downloading all of the files[2].

Hint:Learning to use a command-line interface[3] can be very empowering!.
